5. Movie Recommendation System Using Hybrid Learning

Problem:
Streaming platforms must recommend relevant movies to users based on viewing behavior.

Dataset: MovieLens Dataset

Tasks:
* Load user-movie rating data and construct a user-item matrix.
* Apply unsupervised learning (Matrix Factorization or K-Means) to discover latent user
preference groups.
* Train supervised models to predict user rating for unseen movies:
* Gradient Boosting
* Neural Networks.
* Implement a hybrid recommendation system combining clustering with rating
prediction.

Evaluate using:
* RMSE
* Precision
* Recall
